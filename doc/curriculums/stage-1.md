# 《財商素養課程 \- 第一階段》

# 課程概要

# （線上、雙師模式、15 堂、無基礎）

# **課程目的：**

* 建立正確的金錢價值觀，學會區分「需要」與「想要」的差異。   
* 理解貨幣的演變歷史、現代支付方式及數位貨幣的發展趨勢。   
* 認識收入來源，理解家庭的開銷與賺錢的辛勞。  
* 解析行銷手法與定價策略，看穿折扣、「免費」與廣告背後的商業邏輯。  
* 培養辨識消費陷阱與網購風險的能力，養成理性且聰明的消費決策習慣。

# **學習元素：**

* 貨幣演變與現代金融  
  * 貨幣歷史：從以物易物、金銀到紙幣的防偽技術。  
  * 未來趨勢：認識數位貨幣與電子支付工具。  
  * 支付實務：發票的功能與樣式、付錢的各種方式。  
* 家庭財務與收入來源  
  * 收入來源：認識主動收入與被動收入。  
  * 家庭開銷：理解水電瓦斯與居住等家庭必要開銷。  
* 商業行銷與定價秘密  
  * 行銷陷阱：破解店家折扣、套餐設計與動線規劃的秘密。  
  * 免費經濟：理解免費 App 的策略與免費服務的代價  
  * 價格機制：薄利多銷、供需法則與品牌對價格的影響。  
* 聰明消費與網購安全  
  * 消費決策：辨別「需要」還是「想要」、公益與儲蓄的概念。  
  * 理性判斷：理解 CP 值的意義，避免衝動購物與賭博心態。  
  * 網購識讀：辨識廣告誘人手法、個資安全與限時特價的心理戰。

# **課程對象：**

* 2 年級至 6 年級的學生，無經驗可。

# **課程時數：**

* 50 分鐘 x 15 堂，共 750 分鐘。

# **課程大綱：**

| 堂次 | 課程名稱 | 學習元素 |
| :---: | :---: | ----- |
| 1 | 支出與公益 | 引導學生區分消費時的「需要」與「想要」 探討記帳的好處與個人財務管理基礎 啟發孩子參與公益活動的意識 學習如何評估與挑選合適的公益機構 |
| 2 | 古代貨幣 | 認識古代社會以物易物的交易媒介 探索歷史上不同形式的貨幣演變 理解金與銀作為貨幣的價值基礎 介紹古代紙幣的防偽設計智慧 |
| 3 | 現代貨幣與支付工具 | 認識現代流通貨幣的特徵與功能 學習現代鈔票的防偽技術 認識各種支付工具 介紹行動支付的便利性與使用原理 |
| 4 | 發票與未來貨幣 | 理解索取發票對稅務與保障消費權益的意義 辨識各種形式的統一發票 介紹數位貨幣的類型 探索未來貨幣形式與支付趨勢 |
| 5 | 家庭必要開銷 | 分析家庭日常支出的流向 認識水費、電費與瓦斯費等基本生活開銷 了解租房與買房會付出的花費 認識維持居家環境整潔的隱形成本 |
| 6 | 家庭支出的選擇 | 理解家庭休閒娛樂與旅遊背後的開銷 學習如何在有限預算下做出家庭支出選擇 認識教育投資的重要性與相關費用 探討購車、養車與飼養寵物的長期財務考量 |
| 7 | 錢從哪裡來 | 區分主動收入與被動收入 了解家裡的收入來源 引導孩子思考適合其年齡的合法賺錢方式 討論紅包等不固定收入，該如何謹慎規劃 |
| 8 | 主動收入 | 了解未來職業的多樣性 比較正職工作與兼職工作的差異與優缺點 探討職業選擇時應考量的各種因素 分析創業當老闆的風險、機會與獲利模式 |
| 9 | 店家折扣揭密 | 破解店家打折促銷背後的行銷邏輯 觀察哪些商品鮮少打折及其背後原因 認識購物節檔期的操作手法 解析「尾數 9」定價策略對消費心理的影響 |
| 10 | 店家小巧思 | 分析餐廳菜單排版如何引導顧客點餐 理解套餐組合的定價陷阱與銷售策略 探討收銀台旁商品擺設的衝動消費陷阱 揭開店家動線規劃與視線高度的陳列秘密 了解商家如何透過免費策略獲利 |
| 11 | 生活經濟學 | 探討針對高消費族群設計的高價商品 介紹群眾募資的概念與運作方式 認識共享經濟的優勢與挑戰 解析免費應用程式的獲利來源 引導思考享受「免費」服務時所付出的代價 |
| 12 | 價格競爭 | 理解「薄利多銷」的優勢與風險 認識促銷商品與引流商品的設定 探討無形服務的定價標準 比較不同類型的店家定價策略 |
| 13 | 影響商品的價格因素 | 學習掌握商品價格波動的時機以節省開支 分析品牌價值如何提升商品售價 培養購物前的比價習慣 介紹供需法則對價格的影響 |
| 14 | 網路購物 | 學習看穿網路廣告誇大與誘人的行銷手法 探討網購便利性可能導致的超支問題 破解「限時特價」造成的心理壓力與衝動消費 解析網路購物常見的詐騙手法 |
| 15 | 聰明消費 | 介紹 CP 值的概念 引導思考扭蛋這種機率型商品背後的賭博性質 討論「大家都有我也要有」的同儕壓力與消費行為 學習面對昂貴物品的延遲享樂與規劃 了解生活中常見的訂閱制 |

(為達最佳學習成效，課程大綱將不定期進行修改)
