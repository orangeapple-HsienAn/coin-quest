# 視覺規範 (Visual Specification)

## 0. 設計風格基調

- **風格**：溫馨、遊戲化 (Gamified)、扁平化 (Flat UI) 但帶有輕微立體感
- **視覺感受**：親切、低門檻的理財介面，類似《記帳城市》或日系手帳 App
- **形狀**：大量使用圓角矩形，幾乎沒有尖銳直角
- **背景**：淺橘色底 (`#FFF5EC`) + 橘色格紋 (`rgba(245, 166, 35, 0.15)`)
- **區塊邊框**：所有區塊統一使用 3px 邊框
- **配色**：高明度、低飽和度的「馬卡龍色系」

## 2. 字型系統

### 2.1 字型家族
- 主要字型：系統預設無襯線字型（Noto Sans TC / PingFang TC / Microsoft JhengHei）
- 數字字型：等寬數字，用於金額顯示

### 2.2 字型大小
| 層級 | 大小 | 字重 | 用途 |
|------|------|------|------|
| H1 | 32px | Bold | 資產總額數字 |
| H2 | 24px | Bold | 頁面標題 |
| H3 | 20px | SemiBold | 區塊標題 |
| Body | 16px | Regular | 一般內文 |
| Caption | 14px | Regular | 說明文字 |
| Small | 12px | Regular | 時間戳記、輔助資訊 |

### 6.2 卡片

#### 任務卡片
- 背景：白色
- 邊框：1px solid `#F0E6D8`
- 圓角：16px
- 陰影：卡片陰影

#### 資產區塊卡片
- 背景：功能色（依類型）
- 圓角：12px
- 內有箭頭連結圖示

#### 資產總覽區塊（首頁右側）

**頂部資訊欄**：
- 佈局：左右兩塊圓角矩形，比例約 7:3
- 左側卡片（資產總額）：奶油黃 `#FCD667` 實心背景 + 3px 邊框
- 右側卡片（排名）：軟橘色 `#FFAD60` 實心背景 + 3px 邊框
- 數值字體：Condensed 無襯線字體

**主要內容卡片**（圓餅圖 + 列表在同一卡片內）：
- 容器：白色圓角卡片，帶奶油黃 `#FCD667` 3px 邊框
- 圖表類型：半圓形圓餅圖 (Semi-circle Donut Chart)
- 扇形之間有白色間隙分隔
- 文字：白色文字直接壓在色塊上（數值為 0 時不顯示）

**列表選單區**（在主要內容卡片內）：
- 背景：極淡的功能色背景
- 邊框：3px 實色邊框（顏色對應左側圖示）
- 圓角：14px
- 陰影：輕微底部陰影，製造按鈕凸起感
- 左側：圓形圖示背景（金幣、圖表、小豬、盾牌）
- 右側：右上箭頭 icon (↗)

### 6.3 標籤 (Tag)

## 7. 圖示與插畫

### 7.1 功能圖示
- 風格：圓潤、填充式
- 大小：24px（標準）、32px（強調）
- 顏色：跟隨功能色或中性灰

### 7.2 數值圖示

#### 金幣圖示 (CoinIcon)
- 類型：自訂 SVG 元件 (`/src/components/icons/CoinIcon.tsx`)
- 外觀：金黃色圓形硬幣，中央有星星圖案
- 漸層色：
  - 頂部：`#FFE066`
  - 中部：`#F5C842`
  - 底部：`#D4A017`
- 內圈邊框：`#E8C547`
- 陰影：底部 `#D4A017` 圓形製造立體感
- 預設大小：24px（可透過 `size` prop 調整）
- 使用處：Header 金幣數、任務獎勵、簽到日薪、各頁面金額顯示

## 8. 頁面佈局

### 8.1 Header
- 高度：60px
- 背景：主色橘黃
- 左側：Logo「橘子蘋果」
- 右側：經驗值、金幣數量、使用者頭像與暱稱

### 8.2 首頁佈局
```
┌─────────────────────────────────────────┐
│                 Header                   │
├───────────────────┬─────────────────────┤
│                   │     資產總額 + 排名    │
│    今日任務        ├─────────────────────┤
│                   │      圓餅圖           │
├───────────────────┼─────────────────────┤
│    簽到按鈕        │    可用現金 →        │
├───────────────────┤    投資理財 →        │
│                   │    銀行儲蓄 →        │
│    進入課程        │    持有保險 →        │
│                   │                     │
└───────────────────┴─────────────────────┘
```

### 8.3 內頁佈局
- 左上返回箭頭
- Header 保持顯示
- 內容區域置中，最大寬度 1200px
- 背景：米色 `#FFF9F0`

## 9. 動效規範

### 9.1 過渡時間
| 類型 | 時間 |
|------|------|
| 快速 | 150ms |
| 標準 | 250ms |
| 緩慢 | 350ms |

### 9.2 緩動函數
- 標準：`ease-in-out`
- 進入：`ease-out`
- 離開：`ease-in`

### 9.3 互動動效
- 按鈕 hover：輕微上移 + 陰影加深
- 卡片 hover：陰影加深
- 任務完成：打勾動畫
- 金幣獲得：數字跳動 + 金幣飄入動畫
- 風險卡：翻牌動畫
