# 📊 財商學習平台 — 上線規劃

> **預計上線：7/10（6 週、約 30 工作天）**

平台功能已完成 90%，剩下「同步機制、帳號系統、設計對齊、上線測試」。**課程內容隨 CMS 進度自動上線，不卡工程**。

---

## 🎯 上線交付清單

### 同步機制
- Admin 後台「同步 CMS」按鈕（無需工程師介入）
- 支援三語（zh/en/ja）課程資料 + 遊戲 HTML 同步
- 增量同步（只更新改過的內容）

### 帳號系統（M1 過渡方案）
- Email + 密碼登入（Firebase Auth 原生）
- **先 seed 幾個測試帳號**供內部試用
- 既有匿名帳號清理
- ⚠️ **正式帳號機制延後決策**（SSO / 自助註冊 / Admin 建帳號），上線前必須拍板

### 課程體驗
- 課程主題解鎖機制（依等級 / 前課完成）
- 章節列表完成狀態視覺化（綠色已完成）
- 吉祥物導引下一個應做章節
- 「小議題 / 議題測驗」新章節類型

### 每日測驗（首頁任務）
- 題庫聚合（所有小測驗 + 小議題測驗）
- 每日隨機抽 5 題、標註第幾課第幾單元
- 未解鎖內容於詳解標註「後面才會上到」
- 獎勵：基礎 200 + 答對 ×20（最高 300）+ 鼓勵文案
- ⚠️ 「未上過」判斷依賴解鎖機制（需先完成 A3）

### 設計與多語
- 全平台 UI 對齊 Figma 設計稿（pixel-perfect）
- 語言切換 UI（zh / en / ja）
- 隱私權政策 + 服務條款頁面
- 效能優化（首屏載入加速）
- **平板支援**（網頁版需可在 iPad / Android 平板瀏覽器使用）

### 維運與品質
- 連續簽到獎勵
- 股市休市日邏輯處理
- 完整上線前測試（端對端流程、跨裝置、金錢守恆）
